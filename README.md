## Overview
This repository contains a set of basic Python programs developed as part of programming practice and logic-building exercises.

The objective of these exercises is to strengthen understanding of core Python concepts and improve problem-solving skills through simple, structured programs.

## Programs Included

**1. User Login Check**  
A program that validates username and password credentials.

**2. Pass / Fail Analyzer**  
Analyzes student marks and counts the number of pass and fail results.

**3. Simple Data Cleaner**  
Processes user names by removing extra spaces and converting them to lowercase.

**4. Message Length Analyzer**  
Calculates message lengths and identifies longer messages.

**5. Error Message Detector**  
Counts the number of error entries from a list of log messages.

**6. Unique Word Identification**  
Extracts unique words from a sentence.

**7. Salary Comparison**  
Determines the highest salary from employee data.

**8. Maximum and Minimum Value Detection**  
Finds the largest and smallest values from a list.

**9. Attendance Percentage Calculation**  
Computes attendance percentage based on presence records.

**10. Duplicate Removal**  
Removes duplicate values using basic data structures.

## Concepts Practiced
- Conditional statements (if / else)
- Loops (for)
- Lists, tuples, and sets
- Dictionaries
- String manipulation
- Basic data processing

## Tools Used
- Python
- Jupyter Notebook
